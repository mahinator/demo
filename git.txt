1. Подключитесь к серверу по SSH через Termius:
    ssh masha@192.168.1.135

  2. Склонируйте предоставленный репозиторий в домашнюю папку пользователя  masha  на сервере:
    cd ~
    git clone <URL_ПРЕДОСТАВЛЕННОГО_РЕПОЗИТОРИЯ> exam_project

  3. Перейдите в папку склонированного проекта:
    cd exam_project

  4. Скопируйте все рабочие файлы сайта из директории веб-сервера в текущую папку:
    cp -r /var/www/html/* .

  5. Загрузите файлы документации с локального компьютера (отчеты  .docx , блок-схема и ER-диаграмма  .pdf ) в
  созданную папку  ~/exam_project/  на сервере с помощью вкладки SFTP в Termius.
  6. Отправьте все файлы в репозиторий:
    # Добавляем все скопированные файлы
    git add .

    # Создаем коммит
    git commit -m "Final deploy of code and docs directly from server"

    # Отправляем на сервер контроля версий
    git push origin main
